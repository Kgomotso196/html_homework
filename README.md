############################################################
# WELCOME TO YOUR FIRST WEB PAGE! 🌟
############################################################

Hi there! This guide will help you create your very first web page on Windows, 
where you tell the world your name. Don’t worry if you’ve never coded before; just follow these simple steps.

############################################################
# STEP 1: OPEN COMMAND PROMPT
############################################################

Command Prompt is a tool where you type instructions to your computer.

1. Press `Win + R` on your keyboard
2. Type `cmd` and press Enter
3. A black window will open – this is Command Prompt

############################################################
# STEP 2: CLONE THIS REPOSITORY
############################################################

Cloning means making a copy of this project on your computer.

1. In Command Prompt, type this command and press Enter:

   git clone https://github.com/Kgomotso196/html_homework.git

2. You will now see a folder called `html_homework` in your current location

############################################################
# STEP 3: GO INTO YOUR PROJECT FOLDER
############################################################

1. In Command Prompt, type this and press Enter:

   cd html_homework

This moves you into the folder where your project is stored.

############################################################
# STEP 4: CREATE YOUR FIRST WEB PAGE
############################################################

1. Open File Explorer and go to the `html_homework` folder
2. Right-click inside the folder → New → Text Document
3. Name it: `index.html` (make sure it ends with `.html` NOT `.txt`)
4. Right-click `index.html` → Open with → Notepad

############################################################
# STEP 5: WRITE YOUR FIRST HTML
############################################################

Copy and paste this into Notepad:

   <!DOCTYPE html>
   <html>
     <head>
       <title>My First Page</title>
     </head>
     <body>
       <h1>Hello! My name is [Your Name]</h1>
     </body>
   </html>

- Replace `[Your Name]` with your real name

############################################################
# STEP 6: SAVE YOUR FILE
############################################################

- Press `Ctrl + S` to save your work

############################################################
# STEP 7: OPEN YOUR PAGE IN A BROWSER
############################################################

1. Go to your `index.html` file in File Explorer
2. Double-click it
3. Your web page will open in your browser and show your name 🎉

############################################################
# STEP 8: SAVE YOUR WORK WITH GIT
############################################################

1. Go back to Command Prompt in your project folder
2. Type these commands **one by one**, pressing Enter after each:

   git add index.html
   git commit -m "Add my first page with my name"
   git push

- This will save your changes online (on GitHub)

############################################################
# STEP 9: CELEBRATE!
############################################################

You just created your first web page and saved it with Git. High five! ✋

Now you can try changing the text, adding colors, or making more pages. 🚀
